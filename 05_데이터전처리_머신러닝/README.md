# 05_데이터전처리_머신러닝

## 📚 학습 목표
실제 데이터의 결측치와 이상치를 처리하고, 기본적인 머신러닝 기법을 적용할 수 있다.

## 📁 파일 구성
- `5st.R`: 다섯 번째 수업 코드 (데이터 전처리 및 머신러닝)
- `day5/`: Day 5 상세 실습 파일들
  - `day5_script1.R`: 결측치 및 이상치 처리
  - `day5_script(lm_rpart).R`: 선형회귀 및 의사결정트리
  - `day4_script0.R`: 전처리 기초
  - `day4_script1(35_44).R`: 특정 범위 데이터 처리

## 🎯 주요 학습 내용
1. **결측치(Missing Value) 처리**
   - `is.na()`: 결측치 확인
   - `na.omit()`: 결측치 제거
   - `na.rm = TRUE`: 함수에서 결측치 제외
   - `filter(!is.na())`: dplyr로 결측치 필터링

2. **이상치(Outlier) 처리**
   - `boxplot()`: 박스플롯으로 이상치 탐지
   - 사분위수 기반 이상치 경계값 설정
   - `ifelse()`로 이상치를 NA 처리
   - 도메인 지식 기반 이상치 정의

3. **선형회귀분석(Linear Regression)**
   - `lm()`: 선형회귀 모델 생성
   - `summary()`: 회귀분석 결과 해석
   - `predict()`: 예측값 생성
   - 신뢰구간과 예측구간
   - 잔차 분석 및 모델 진단

4. **의사결정트리(Decision Tree)**
   - `rpart()`: 의사결정트리 모델
   - `rpart.plot()`: 트리 시각화
   - 데이터 분할: 훈련/테스트 세트
   - 모델 평가: 혼동행렬, 정확도

5. **모델 평가**
   - `createDataPartition()`: 계층적 샘플링
   - `confusionMatrix()`: 분류 성능 평가
   - 교차검증과 과적합 방지

## 🔧 주요 패키지
- **caret**: 머신러닝 통합 패키지
- **rpart**: 의사결정트리
- **rpart.plot**: 트리 시각화
- **dplyr**: 데이터 전처리
