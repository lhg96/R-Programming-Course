# 01_기초문법

## 📚 학습 목표
R의 기본 문법과 데이터 타입을 이해하고 활용할 수 있다.

## 📁 파일 구성
- `1st.R`: 첫 번째 수업 코드 (변수, 벡터, 기본 시각화)
- `day1_script.R`: Day 1 연습 스크립트 (상세한 기초 문법 실습)
- `basic문법`: 기본 문법 정리 파일

## 🎯 주요 학습 내용
1. **변수와 대입 연산자**
   - `<-` 연산자 사용법
   - 변수 타입별 특성

2. **벡터(Vector)**
   - 벡터 생성: `c()`, `seq()`, `:`
   - 벡터 연산 및 조작
   - 인덱싱과 조건부 선택

3. **데이터 타입**
   - 숫자형(Numeric)
   - 논리형(Logical)
   - 문자형(Character)
   - 팩터형(Factor)

4. **기본 함수들**
   - `print()`, `str()`, `length()`
   - `mean()`, `sum()`, `min()`, `max()`

5. **첫 번째 시각화**
   - ggplot2 패키지 소개
   - `qplot()` 함수 활용
