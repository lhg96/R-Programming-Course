# 02_데이터조작

## 📚 학습 목표
데이터프레임을 생성하고 dplyr 패키지를 활용하여 데이터를 효율적으로 조작할 수 있다.

## 📁 파일 구성
- `2st.R`: 두 번째 수업 코드 (데이터프레임, dplyr 기초)
- `day2_script.R`: Day 2 연습 스크립트 (데이터 조작 실습)
- `mission1.R`: 복습 문제 및 미션

## 🎯 주요 학습 내용
1. **데이터프레임(Data Frame)**
   - 데이터프레임 생성 및 구조 이해
   - 행과 열 인덱싱 방법
   - `$` 연산자를 활용한 열 접근

2. **dplyr 패키지**
   - `rename()`: 변수명 변경
   - `filter()`: 행 필터링
   - `select()`: 열 선택
   - `arrange()`: 정렬
   - `mutate()`: 새 변수 생성

3. **조건부 처리**
   - `ifelse()` 함수 활용
   - 다중 조건 처리

4. **파이프 연산자**
   - `%>%` 연산자로 코드 연결
   - 가독성 높은 코드 작성

5. **데이터 요약**
   - `group_by()`, `summarise()` 활용
   - 기초 통계량 계산
