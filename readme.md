# 📊 R 프로그래밍 학습 자료

## 🎓 과정 개요
**R 강의 자료** (2020년 8월 20일)  
R 기초 문법부터 데이터 분석, 시각화, 머신러닝까지 5일간의 집중 학습 과정

## 🛠 환경 설정

### R 및 RStudio 설치
- **R 다운로드**: https://cran.r-project.org/bin/windows/base/R-4.0.2-win.exe
- **RStudio 다운로드**: https://rstudio.com/products/rstudio/download/
- **RStudio Cloud**: https://rstudio.cloud/ (설치가 어려운 경우)

## 📁 폴더 구조

```
R/
├── 01_기초문법/              # Day 1: R 기본 문법과 데이터 타입
├── 02_데이터조작/            # Day 2: 데이터프레임과 dplyr
├── 03_기본시각화/            # Day 3: plot 함수와 기본 차트
├── 04_고급시각화/            # Day 4: ggplot2 고급 기능
├── 05_데이터전처리_머신러닝/  # Day 5: 전처리 및 ML 기초
├── datasets/                # 실습용 데이터셋
├── outputs/                 # 생성된 그래프 및 결과물
├── R_학습정리.md            # 전체 학습 내용 정리
└── README.md               # 이 파일
```

## 📚 학습 진도

### 🔰 [Day 1: 기초 문법](01_기초문법/)
- 변수와 대입 연산자
- 벡터 생성과 조작
- 데이터 타입 (숫자, 논리, 문자, 팩터)
- 기본 함수와 첫 번째 시각화

### 🔧 [Day 2: 데이터 조작](02_데이터조작/)
- 데이터프레임 생성 및 조작
- dplyr 패키지 활용
- 조건부 처리와 파이프 연산자
- 데이터 필터링, 정렬, 요약

### 📊 [Day 3: 기본 시각화](03_기본시각화/)
- 기본 plot 함수
- 파이차트, 막대차트, 히스토그램
- ggplot2 기초 문법
- 그래프 꾸미기와 테마

### 🎨 [Day 4: 고급 시각화](04_고급시각화/)
- ggplot2 고급 기능
- 다양한 geom 함수들
- 그래프 레이아웃과 테마
- 색상, 스타일링, 범례

### 🤖 [Day 5: 데이터 전처리 & 머신러닝](05_데이터전처리_머신러닝/)
- 결측치와 이상치 처리
- 선형회귀분석
- 의사결정트리
- 모델 평가와 예측
